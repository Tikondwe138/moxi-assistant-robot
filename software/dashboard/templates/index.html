Voice Commands

You can chat with MOXI using the following commands:

  • "Hello" or "Hi Moxi" Greets the assistant and triggers a wave.
  • "Find contact [Name]" e.g., "Find contact John". Looks up phone number.
  • "Check inventory for [Item]" e.g., "Check inventory for Arduino". Checks stock availability.
  • "Execute [gesture] gesture" e.g., "Execute point gesture". Triggers robot action.
  • "Exit" or "Shut down" Safely shuts down the MOXI system.
